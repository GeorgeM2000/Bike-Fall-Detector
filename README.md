# Εφαρμογή ανίχνευσης πτώσης
Στόχος της εφαρμογής είναι να ανιχνεύει ατυχήματα και πτώσεις μηχανής η ποδηλάτου και να ειδοποιήσει επιλεγμένες απο τον χρήστη επαφές μέσω μηνυμάτων SMS. 

# Περιγραφή 
Η εφαρμογή έχει υλοποιηθεί για την πλατφόρμα Android χρησιμοποιώντας τη γλώσσα προγραμματισμού java. Επίσης, χρησιμοποιείται σαν βάση δεδομένων η Firebase της Google, η οποία είναι εύχρηστη και συνδέεται πολύ εύκολα με οποιαδήποτε Android εφαρμογή. Στην βάση δεδομένων αποθηκεύονται πληροφορίες χρηστών, οπως στ οποίες θα σταλθούν οι ειδοποιήσεις(SMS) εάν συμβεί ένα ατύχημα
